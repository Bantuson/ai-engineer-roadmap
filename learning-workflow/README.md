# Learning Workflow: 3-Week AI Engineer Intensive

A structured 3-week learning system (January 19 - February 7, 2026) designed to accelerate mastery of AI engineering skills through disciplined daily practice.

## Program Overview

| Aspect | Details |
|--------|---------|
| **Duration** | 20 days (Jan 19 - Feb 7, 2026) |
| **Daily Hours** | 13h weekdays (8am-9pm), 6h weekends |
| **Total Hours** | ~150 hours |
| **Focus Areas** | Python, AI Frameworks, PM, DSA, Prompts |

## Quick Start

1. **Read the roadmap**: Start with `overview/3-week-roadmap.md`
2. **Check today's plan**: Navigate to `week{N}/day{NN}-{day}.md`
3. **Complete drills**: Open `prompt-drills/week{N}/day{NN}-drills.md`
4. **Track progress**: Update `trackers/daily-log-template.md`

## Week-by-Week Focus

### Week 1: Foundation (Jan 19-25)
- Python Days 1-15 review
- CrewAI + LlamaIndex deep dive
- PM Foundations
- DSA: Arrays, strings, hashmaps (15 problems)
- 84 prompts

### Week 2: Deep Dive (Jan 26 - Feb 1)
- Python Days 16-25
- LangGraph, LangChain, OpenAI SDK, Pydantic AI
- PM Technical + AI modules
- DSA: Trees, graphs, lists (20 problems)
- 84 prompts

### Week 3: Integration (Feb 2-7)
- Python Days 26-30 + consolidation
- Google ADK, AutoGen, Semantic Kernel, Claude SDK
- PM Prompt Engineering + Capstone
- DSA: Recursion, DP (15 problems)
- 72 prompts

## Success Criteria

By February 7, 2026:
- [ ] 240 total prompts written
- [ ] 50+ DSA problems solved
- [ ] Eval system for 5+ mini-projects
- [ ] Red team report for 3+ projects
- [ ] PM Capstone complete
- [ ] Interview prep materials ready

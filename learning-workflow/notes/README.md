## How to Use

1. **Before each day**: Review the day's plan
2. **During the day**: Take notes in the appropriate sections
3. **End of day**: Complete the reflection section
4. **Weekly**: Review notes to identify patterns

## Note Template Sections

Each note file has sections for:
- **Morning Session**: Prompt drills, reading, Python
- **Afternoon Session**: Mini-project, PM course, DSA
- **Evening Session**: Continued work, reflection
- **Key Takeaways**: Most important learnings
- **Questions**: Things to clarify or research

## Best Practices

- Be concise but thorough
- Include code snippets when helpful
- Note what worked and what didn't
- Record questions for later research
- Celebrate wins, no matter how small

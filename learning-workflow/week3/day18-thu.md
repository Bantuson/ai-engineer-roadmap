## ReAct Pattern

```
THOUGHT → ACTION → OBSERVATION → THOUGHT → ... → ANSWER

Example:
THOUGHT: "I need to read the file first"
ACTION: bash: cat script.py
OBSERVATION: [file contents]
THOUGHT: "I see a bug on line 15"
ANSWER: "Found IndexError risk..."
```

# Module 01: Product Management Foundations

Welcome to the foundations module. These six lessons establish the core concepts every product manager needs, regardless of specialization.

## Module Overview

| Lesson | Topic | Key Concepts |
|--------|-------|--------------|
| 01 | What is PM? | Role definition, PM types, skills |
| 02 | Product Lifecycle | Introduction, growth, maturity, decline |
| 03 | User Research & Discovery | Research methods, continuous discovery |
| 04 | Product Strategy | Vision, positioning, competitive analysis |
| 05 | Roadmapping | Roadmap types, prioritization basics |
| 06 | Metrics & Analytics | KPIs, North Star, retention, unit economics |

## Learning Objectives

By completing this module, you will be able to:

- [ ] Explain the PM role and how it differs across company types
- [ ] Identify which lifecycle stage a product is in and adjust strategy accordingly
- [ ] Conduct user research interviews and synthesize findings
- [ ] Develop a product strategy with clear positioning
- [ ] Create outcome-based roadmaps that align stakeholders
- [ ] Define and track meaningful product metrics

## Prerequisites

- None! This module is designed for beginners
- Helpful: Any experience using software products critically

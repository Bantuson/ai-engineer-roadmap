### Product Management Tools

Purpose-built tools for product work.

#### Productboard

**Best for:** Customer-centric product teams, feedback aggregation

**Key features:**
- Customer feedback portal
- Feature prioritization matrix
- Roadmap views (multiple formats)
- Objectives/key results tracking
- Integrations with Slack, Intercom, Zendesk

**Pricing:** From ~$20/user/month

**Use when:** You need to centralize customer feedback and connect it to roadmap decisions.

#### Aha!

**Best for:** Enterprise PM teams, strategy-first approach

**Key features:**
- Strategic planning (goals → initiatives → features)
- Multiple product line support
- Advanced roadmapping
- Idea management
- Extensive customization

**Pricing:** From ~$59/user/month

**Use when:** You need comprehensive strategic planning and work in larger organizations.

#### Airfocus

**Best for:** Prioritization-focused teams

**Key features:**
- Prioritization frameworks built-in (RICE, custom)
- Roadmap views
- Modular approach (add features as needed)
- Integrations with dev tools

**Pricing:** From ~$15/user/month

**Use when:** Prioritization is your main challenge and you want focused tooling.

### Project/Task Management

Where day-to-day work gets tracked.

#### Linear

**Best for:** Engineering-centric product teams, modern startups

**Key features:**
- Fast, keyboard-driven interface
- Cycles (sprints) and projects
- GitHub integration
- Roadmap views
- AI features for issue creation

**Pricing:** From ~$8/user/month

**Why it's popular:** Speed, beautiful design, engineer-friendly. The tool modern teams often choose.

#### Jira

**Best for:** Large organizations, complex workflows, enterprise

**Key features:**
- Highly customizable workflows
- Extensive integrations
- Advanced reporting
- Scaled agile framework support
- Confluence integration

**Pricing:** From ~$8/user/month (cloud)

**Why it persists:** Enterprise features, existing investment, extensive customization.

#### Asana

**Best for:** Cross-functional teams, non-engineering work

**Key features:**
- Multiple views (list, board, timeline, calendar)
- Goals and portfolios
- Forms and intake
- Workload management

**Pricing:** From ~$11/user/month

**Use when:** You need project management that works for non-engineering stakeholders too.

#### Shortcut (formerly Clubhouse)

**Best for:** Mid-size product teams wanting Jira alternative

**Key features:**
- Stories, epics, milestones
- Iteration planning
- API-first approach
- Clean interface

**Pricing:** From ~$8.50/user/month

**Use when:** You want something more powerful than Linear but simpler than Jira.

### Documentation Tools

Where knowledge lives.

#### Notion

**Best for:** Flexible, all-in-one workspace

**Key features:**
- Blocks-based editing
- Databases with multiple views
- Wikis and docs in one place
- Templates and AI assistance
- Easy collaboration

**Pricing:** From ~$8/user/month

**Why it dominates:** Flexibility to be whatever you need—PRDs, wikis, project tracking, all in one.

#### Confluence

**Best for:** Atlassian shops, enterprise documentation

**Key features:**
- Deep Jira integration
- Page trees and spaces
- Permissions and compliance
- Templates and macros

**Pricing:** From ~$6/user/month (cloud)

**Use when:** Already using Jira, need enterprise features.

#### Coda

**Best for:** Document-as-app use cases

**Key features:**
- Docs that function like apps
- Tables with formulas
- Packs (integrations)
- Automations

**Pricing:** From ~$10/user/month

**Use when:** You want documents that can do more than static text.

### Analytics Tools

Understanding user behavior.

#### Amplitude

**Best for:** Product-led growth, self-serve analytics

**Key features:**
- Event-based tracking
- Funnels and cohorts
- User paths and journeys
- Behavioral targeting
- Experiment analysis

**Pricing:** Free tier available; paid from ~$61k/year

**Use when:** You need deep behavioral analytics and have product-led growth model.

#### Mixpanel

**Best for:** Event tracking, mobile and web

**Key features:**
- Real-time analytics
- User profiles
- A/B testing integration
- Alerts and reporting

**Pricing:** Free tier available; paid varies

**Use when:** You need flexible event analytics with good mobile support.

#### PostHog

**Best for:** Open-source preference, all-in-one

**Key features:**
- Product analytics
- Session recording
- Feature flags
- A/B testing
- Self-host option

**Pricing:** Free tier; paid scales with usage

**Use when:** You want multiple tools in one and prefer open source.

#### Heap

**Best for:** Automatic event capture

**Key features:**
- Auto-capture (no manual tagging)
- Retroactive analysis
- Journey mapping
- Conversion analysis

**Pricing:** Varies by volume

**Use when:** You want analytics without heavy implementation.

### Design Collaboration

Working with design teams.

#### Figma

**Best for:** Design files, prototyping, handoff

**Key features:**
- Real-time collaboration
- Component libraries
- Prototyping
- Developer handoff (Dev Mode)
- Plugins ecosystem

**Pricing:** Free tier; paid from ~$15/user/month

**Industry standard:** Most design teams use Figma. PMs should be comfortable navigating and commenting.

#### FigJam

**Best for:** Whiteboarding, workshops

**Key features:**
- Collaborative whiteboard
- Templates for workshops
- Sticky notes, voting, timers
- Integrated with Figma

**Pricing:** Included with Figma

**Use when:** Brainstorming, retrospectives, user journey mapping, any collaborative thinking.

#### Miro

**Best for:** Extensive whiteboarding needs

**Key features:**
- Large canvas
- Many templates
- Video chat built-in
- Integrations

**Pricing:** Free tier; paid from ~$8/user/month

**Use when:** You need more whiteboarding power than FigJam or prefer standalone tool.

### User Research Tools

Understanding users directly.

#### Dovetail

**Best for:** Research repository, synthesis

**Key features:**
- Transcription and tagging
- Insight repository
- Pattern identification
- Team collaboration on research

**Pricing:** From ~$29/user/month

**Use when:** You do regular user research and need to organize findings.

#### UserTesting

**Best for:** Remote usability testing

**Key features:**
- Participant recruitment
- Video recording
- Task-based testing
- Highlight reels

**Pricing:** Enterprise pricing

**Use when:** You need usability testing at scale.

#### Hotjar / FullStory

**Best for:** Session recording, heatmaps

**Key features:**
- Session replay
- Heatmaps and click maps
- Feedback widgets
- Funnel analysis

**Pricing:** Freemium; paid varies

**Use when:** You want to see how users actually interact with your product.

#### Sprig

**Best for:** In-product research

**Key features:**
- In-app surveys
- User targeting
- Concept testing
- AI-powered analysis

**Pricing:** Varies

**Use when:** You want to ask users questions in context.

### AI Tools for PMs

Productivity multipliers.

#### Claude / ChatGPT / Gemini

**Best for:** Writing, research, analysis

**Use for:**
- Drafting PRDs and documentation
- Synthesizing research
- Competitive analysis
- Brainstorming
- Data interpretation

**See:** [04-Prompt Engineering module](../04-prompt-engineering-pm/) for detailed usage.

#### GitHub Copilot

**Best for:** PMs who write code/scripts

**Use for:**
- Quick data analysis scripts
- Automations
- Understanding technical code

#### AI Features in Existing Tools

Most tools now include AI:
- **Linear:** Auto-issue creation
- **Notion:** AI writing and Q&A
- **Amplitude:** AI-powered insights
- **Figma:** AI design suggestions

### Tool Selection Framework

When choosing tools:

#### 1. Assess Your Needs

- What problems are you solving?
- What's your team size and budget?
- What tools do adjacent teams use?
- What integrations matter?

#### 2. Evaluate Options

| Criterion | Weight | Tool A | Tool B |
|-----------|--------|--------|--------|
| Core features | High | | |
| Integrations | Medium | | |
| Ease of use | High | | |
| Price | Medium | | |
| Scalability | Low | | |

#### 3. Try Before Committing

- Use free trials fully
- Test with real workflows
- Get team feedback
- Check migration difficulty

#### 4. Plan for Integration

- How does data flow between tools?
- What manual work is needed?
- Are APIs available?

### Common Tool Stacks

- Linear + Notion + Figma + Amplitude + Slack

**Enterprise stack:**
- Jira + Confluence + Figma + Adobe Analytics + Teams

**Product-led growth stack:**
- Productboard + Linear + Notion + Amplitude + Figma + PostHog

## Key Takeaways

1. **The PM tool stack spans: product management, project tracking, documentation, analytics, design, research, and AI**
2. **Linear and Notion dominate modern product teams; Jira and Confluence persist in enterprise**
3. **Analytics tools (Amplitude, Mixpanel, PostHog) are essential for data-driven PM**
4. **AI tools (Claude, ChatGPT) are now productivity multipliers for PM work**
5. **Choose tools based on needs, integrations, and what adjacent teams use**
6. **Tools change—invest in skills that transfer across tools**

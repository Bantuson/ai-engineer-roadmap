# Module 05: Frameworks & Tools

Practical frameworks and modern tools for effective product management.

## Module Overview

| Lesson | Topic | Application |
|--------|-------|-------------|
| 01 | Prioritization Frameworks | RICE, MoSCoW, ICE, Kano, JTBD |
| 02 | Stakeholder Management | Mapping, communication, influence |
| 03 | PM Tools (2026) | Modern toolkit for product managers |
| 04 | Certifications & Paths | Career development resources |

## Learning Objectives

By completing this module, you will be able to:

- [ ] Select and apply the right prioritization framework for different situations
- [ ] Map stakeholders and develop communication strategies for each
- [ ] Navigate the modern PM tool landscape effectively
- [ ] Plan your PM career development and certification path

## Prerequisites

- Completed: [01-Foundations](../01-foundations/) module
- Helpful: Real PM or product-adjacent work experience

## Lessons

### [01 - Prioritization Frameworks](./01-prioritization-frameworks.md)
Master the key frameworks: RICE, MoSCoW, ICE, Kano, and Jobs-to-be-Done. Know when to use each.

### [02 - Stakeholder Management](./02-stakeholder-management.md)
Map your stakeholders, develop communication strategies, and influence without authority.

### [03 - PM Tools (2026)](./03-pm-tools-2026.md)
The modern PM toolkit: project management, documentation, analytics, design collaboration, and AI tools.

### [04 - Certifications & Paths](./04-certifications-paths.md)
PM certification programs, learning paths, and career development resources.

## Time Estimate

2 weeks at 1-2 hours per day

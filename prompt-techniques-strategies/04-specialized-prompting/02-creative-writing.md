# Creative Writing Prompting

## Overview

Creative writing prompts require balancing structure with creative freedom. This module covers techniques for generating stories, content, and creative text while maintaining quality and consistency.

## Story Writing Prompts

### Short Story Generation

```
Write a short story with the following elements:

GENRE: Science fiction
THEME: The cost of immortality
SETTING: Space station orbiting a dying sun, year 3847
PROTAGONIST: Dr. Elena Voss, 847 years old, remembers Earth

PLOT REQUIREMENTS:
- Opening hook: Start with a moment of crisis
- Conflict: Elena must choose between saving one person or many
- Climax: A revelation that changes her perspective
- Resolution: Bittersweet ending with hope

STYLE GUIDELINES:
- Third person limited POV
- Literary but accessible prose
- Emphasis on sensory details
- Dialogue reveals character
- Show, don't tell emotions

LENGTH: 2000-2500 words

AVOID:
- Info dumping in first paragraph
- Deus ex machina solutions
- Purple prose
- Clichéd sci-fi tropes
```

### Character Development

```
Create a detailed character profile:

BASICS:
Name: [generated or provided]
Age: [age]
Role in story: [protagonist/antagonist/supporting]

DEVELOP:

External:
- Physical appearance (3-4 distinctive features)
- Mannerisms and habits
- Speech patterns
- Style of dress

Internal:
- Core motivation (what they want most)
- Fatal flaw
- Secret fear
- Moral code
- Contradictions in personality

Background:
- Formative childhood event
- Greatest achievement
- Biggest regret
- Key relationships

Character arc potential:
- Where they start
- What challenges them
- How they might change

Write a 500-word scene that reveals their character through action,
not exposition.
```

### Dialogue Writing

```
Write a dialogue scene with these specifications:

CHARACTERS:
- Alex: Nervous job applicant, tends to ramble
- Jordan: Experienced interviewer, direct but fair

CONTEXT:
Job interview for a position Alex desperately needs.
Alex is qualified but has a gap in employment history.

GOALS:
- Alex: Get the job, avoid discussing the gap
- Jordan: Assess Alex's fit, understand the gap

REQUIREMENTS:
- Show personality through speech patterns
- Include subtext (what they mean vs. what they say)
- Natural interruptions and reactions
- Tension building and release
- Physical actions/beats between lines
- No dialogue tags except "said" and "asked"

FORMAT:
[Character name]: "[Dialogue]"
[Action/reaction]

LENGTH: 500-700 words
```

## Content Creation Prompts

### Blog Post Writing

```
Write a blog post with these specifications:

TOPIC: [topic]
TARGET AUDIENCE: [who]
GOAL: [inform/persuade/entertain]

STRUCTURE:
1. Hook (question, statistic, or bold statement)
2. Introduction (promise what reader will learn)
3. Main content (3-5 sections with subheadings)
4. Practical takeaways (actionable items)
5. Conclusion with call-to-action

STYLE:
- Conversational but professional
- Active voice
- Short paragraphs (2-3 sentences)
- Use "you" to address reader directly
- Include examples and analogies

SEO ELEMENTS:
- Primary keyword: [keyword]
- Include keyword in: title, first paragraph, one heading
- Meta description (under 160 characters)

LENGTH: 1200-1500 words
```

### Marketing Copy

```
Create marketing copy for:

PRODUCT/SERVICE: [description]
UNIQUE VALUE PROPOSITION: [what makes it special]
TARGET AUDIENCE: [demographics, pain points]
TONE: [professional/casual/luxurious/urgent]

DELIVERABLES:

1. HEADLINE OPTIONS (5):
   - Hook attention
   - Communicate value
   - Under 10 words each

2. TAGLINE:
   - Memorable phrase
   - Under 7 words

3. BODY COPY (150 words):
   - Lead with benefit
   - Address pain point
   - Social proof element
   - Clear call-to-action

4. CTA BUTTON OPTIONS (3):
   - Action-oriented
   - Create urgency or value

AVOID:
- Clichés like "world-class" or "cutting-edge"
- Vague claims without specifics
- Passive voice
- Jargon unfamiliar to audience
```

### Social Media Content

```
Create social media content for:

PLATFORM: [Twitter/LinkedIn/Instagram]
BRAND VOICE: [descriptive adjectives]
TOPIC: [content topic]
GOAL: [engagement/awareness/conversion]

GENERATE:

For Twitter (280 chars):
- 5 tweet options
- Include relevant hashtags
- Hook in first line
- Thread version if needed

For LinkedIn (long-form):
- Opening hook
- Value-driven content
- Professional tone
- Call to engage

For Instagram:
- Caption (under 150 words)
- Emoji usage (appropriate to brand)
- Hashtag set (20-30)
- Story slide ideas (5)

Each should:
- Stop the scroll
- Provide value
- Encourage interaction
- Align with brand voice
```

## Style and Tone Control

### Style Transfer

```
Rewrite this text in the style of [author/style]:

ORIGINAL TEXT:
[text to transform]

TARGET STYLE: [specific author or style description]

STYLE ELEMENTS TO INCORPORATE:
- Sentence structure typical of target
- Vocabulary level and choice
- Rhythm and pacing
- Distinctive techniques
- Thematic approach

MAINTAIN:
- Core meaning and information
- Approximate length

ANALYZE:
Before rewriting, identify 3-5 key characteristics of the
target style that you will incorporate.
```

### Tone Adjustment

```
Adjust this text from [current tone] to [target tone]:

ORIGINAL:
[text]

TONE SPECTRUM:
Formal ←→ Casual
Serious ←→ Playful
Technical ←→ Simple
Distant ←→ Personal
Neutral ←→ Emotional

TARGET POSITION:
[describe where on each spectrum]

SPECIFIC CHANGES:
- Vocabulary adjustments needed
- Sentence structure changes
- Examples to add or remove
- Level of directness
```

## Creative Constraints

### Constrained Writing

```
Write a piece with these creative constraints:

CONSTRAINT TYPE: [choose one or combine]
- Word limit: Exactly 100 words
- Lipogram: Without using letter 'e'
- Anagram: Each sentence starts with next letter of [word]
- Structure: 6-word story
- Time: Story spans exactly 60 seconds

TOPIC: [subject matter]

These constraints should:
- Force creative solutions
- Create interesting effects
- Not feel gimmicky
- Serve the content
```

### Genre Blending

```
Combine these genres in a cohesive piece:

PRIMARY GENRE: [genre 1]
SECONDARY GENRE: [genre 2]

BALANCE: [e.g., "70% primary, 30% secondary"]

REQUIREMENTS:
- Seamlessly integrate elements
- Use conventions of both genres
- Create something fresh, not jarring
- Target audience appreciates both

Example: "Write a noir detective story with fantasy elements"
- Noir: First-person cynical narrator, urban setting, femme fatale
- Fantasy: Magic system, mythical creatures
- Integration: Magic is illegal, detective investigates magical crimes
```

## Quality Control

### Revision Prompts

```
Revise this creative writing piece:

ORIGINAL:
[text]

FOCUS AREAS:
1. SHOW DON'T TELL
   - Convert telling statements to scenes
   - Add sensory details

2. DIALOGUE
   - Make each character sound distinct
   - Remove unnecessary tags
   - Add subtext

3. PACING
   - Vary sentence length
   - Balance scene and summary
   - Identify slow spots

4. VOICE
   - Strengthen narrative voice
   - Remove clichés
   - Enhance unique style

Provide:
- Tracked changes version
- Summary of major improvements
- One unchanged paragraph (best as-is)
```

## Best Practices

1. **Define voice clearly** - Specific style guidance yields better results
2. **Balance structure and freedom** - Too many constraints kill creativity
3. **Iterate and refine** - Creative writing benefits from revision
4. **Include examples** - Show the quality/style you want
5. **Specify audience** - Content should match reader expectations
